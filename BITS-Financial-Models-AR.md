## Model 1: Example Customer Deal (Manufacturing)

### Client Profile

**Company:** Mid-size manufacturer
- Revenue: $15M annually
- Employees: 50 (15 office, 35 factory floor)
- Current AWS spend: $8,500/month
- Pain points: High cloud costs, unreliable infrastructure, no productivity tools

---

### Phase 1: Infrastructure Migration (Original Pitch)

**Current State:**
- AWS EC2 instances: $3,500/month
- AWS S3 storage: $1,200/month
- AWS RDS database: $2,800/month
- Data transfer: $1,000/month
- **Total: $8,500/month = $102,000/year**

**B.I.T.S. Solution:**
- Edge server (on-site): Hardware $6,000 + Monthly $500
- DePIN backup (Akash): $800/month
- B.I.T.S. management: $1,200/month
- **Total: $2,500/month = $30,000/year**

**Savings: $72,000/year**

**ROI on Infrastructure:**
- Upfront: $6,000
- Monthly savings: $6,000
- Payback period: 1 month
- 3-year savings: $216,000

---

### Phase 2: AR Integration (NEW)

**Hardware:**
- 25x Vuzix Z100 @ $1,000 (enterprise cost) = $25,000
- B.I.T.S. markup (25%): $6,250
- **Client pays: $31,250 for hardware**

**Software/Services (One-Time):**

  - AR work instruction system
  - Integration with existing ERP (SAP, etc.)
  - Quality control checklist overlays
  - Production metrics dashboard

- Auki spatial mapping: $10,000
  - Map entire factory floor (50,000 sq ft)
  - Create persistent AR anchors at each workstation
  - Set up local posemesh node on edge server

- IoTeX sensor integration: $8,000
  - 15x Pebble sensors for equipment monitoring
  - AR overlays showing sensor data
  - Smart contract triggers (Zebec integration)

- System integration: $15,000
  - Connect to ERP system

**Total One-Time: $58,000**

**Monthly Recurring:**

- DePIN backup (already included): $0 additional
- AR device management (25 devices @ $50/mo): $1,250

**Client Total Investment:**

- $72,000/year (vs AWS)

- 35 factory workers x $50K avg salary = $1.75M labor cost
- 40% x $1.75M = $700K equivalent value
- Conservative estimate: 20% realized = **$140K/year value**

- 15% reduction = **$15K/year savings**

**Training Time Reduction:**
- Currently: 2 weeks to train new worker = $2,000 per worker
- With AR: 3 days = $600 per worker
- Savings: $1,400 per new hire
- 10 new hires/year = **$14K/year savings**

**Total Value:**
- Infrastructure savings: $72,000
- Productivity gain: $140,000
- Quality improvement: $15,000
- Training reduction: $14,000
- **Total: $241,000/year**

**ROI Calculation:**
- Investment: $89,250 upfront
